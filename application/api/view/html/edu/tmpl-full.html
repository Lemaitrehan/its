二、甲方权利和义务

（一）为乙方获取学历证书提供报名、报考、资料审核、入学测试、录取查询、学籍注册、考试、毕业论文答辩、毕业证申请等相关管理服务工作，包含教育主管部门或高校规定或要求培训机构应尽的服务。

（二）为乙方通过相应考试提供教育主管部门规定的、高等院校规定的相应教学服务。

（三）为乙方通过考试提供国家政策和法规允许提供的教材、教学资料等。

（四）按照教育主管部门规定与要求制定统一的、公开的、公正的学员管理规范并要求乙方遵照执行的权利。

（五）对于未按本合同约定的时间缴纳学费及相关费用的学员，甲方保留随时终止提供后续服务且不予退还任何已收费用的权利，乙方自行承担被终止后续服务导致的所有后果。

（六）因国家政策变动等不可抗力因素导致的成人学历教育的要素发生变化，如：考试形式、学习方式、证书类型、毕业证书获取时间等，均以国家颁布的政策文件为准。乙方不得以此为由要求甲方退还培训费用。

（七）除国家相关法律法规和《广东省民办职业培训机构管理规定》中规定的有关学费退费的情形之外，甲方无需承担其他任何退费义务，乙方也不得以除此以外的任何理由提出退费要求。

 

三、乙方权利和义务

（一）乙方需严格遵守教育主管部门、报读院校和甲方的教学管理相关规定。

（二）乙方应按甲方要求，完成相关学习、考试等，以确保能顺利毕业。乙方在学习有效期内因故不能按要求参加学习和考试的，可以在有效期内向甲方申请并办理停学手续，根据国家教育主管部门相关政策及高校内部管理规定要求，停学时间不得超过两年，乙方停学期间甲方将保留学员档案。如乙方学习有效期间没有参加学习或参加相应考试且没有办理停学手续，视为乙方自动放弃继续参加学习的权利，乙方无权要求甲方退还任何费用，乙方需自行承担全部损失和责任。

（三）乙方应根据要求及时提交与报考、学习、毕业证申办等相关的资料证件，如因乙方未能及时提交，导致影响学籍注册、延期毕业或不能毕业等，责任由乙方自行承担。

（四）乙方缴纳了全部约定费用，到期未能通过毕业考试的，可以以书面方式向甲方申请提供后续不超过两年的报考或补考的增值服务；未提交书面申请的，甲方不予提供增值服务且不承担任何责任。

（五）乙方需按照本合同约定，按期缴纳学费及相关费用。因延期付款而产生的任何后果（如：影响学籍注册、延期毕业及其他相关后果等），均由乙方自行承担，与甲方无关。

（六）乙方已缴纳的报名费及论文指导费费一律不予退还。除报读成教的学员未被录取之外，其他已缴纳的教材费一律不予退费。

（七）乙方有拒绝缴付甲乙双方约定收费事项以外任何费用的权利。

八、本意向书签订后即产生相应法律效力，如乙方后续单方面提出不购买以上课程，乙方已交纳的学费定金不可退还。如乙方变更课程报读意愿，已交纳的学费定金分以下三种情形处理：

1、报读自考的，学费定金可转作其他课程的等额学费款。

2、报读成人高考、网络教育，未被报读院校录取的，学费定金可转作其他课程的等额学费款。

3、报读成人高考、网络教育，已被报读院校录取的，学费定金不转不退。

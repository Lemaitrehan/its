二、甲方权利和义务 

（一）负责学员的教学计划制定及学习期间的管理和学习，并提供相应的学习环境和教学设备，提供具有相应专业能力及教学能力的授课教师。 

（二）根据经双方认可的课程计划安排相应教学活动，按教学管理规定完成学习后向学员颁发结业证书。

（三）甲方只对已交清全款的课程进行学习计划安排和培训，未交清全款的可暂时不做安排及培训。学员自行承担未交清全款的不良后果。

（四）甲方承诺为学习期间能够严格遵守学校各项管理制度并通过考试获得结业或毕业证书的学员提供至少二次推荐就业服务，在乙方毕业后一个月内必须提供二次就业推荐服务；如乙方需要并提出申请，甲方同意在乙方毕业后一个月后至六个月内再次为乙方提供就业推荐服务。

（五）甲方承诺在每个课程开课三次后，对所聘请授课的老师进行学员满意度调查，学员调查满意度达不到80%以上，无条件更换授课教师，直至学员满意度达80%以上为止。

（六）甲方安排的班主任老师在担任该职务时，如有80%以上学员对其工作不认可（需明确提出理由并由学员举手表决），可向甲方提出更换班主任申请，甲方应在收到申请后一周内更换工作人员；除此之外，班主任担任班级管理工作期间无违法违纪行为的，原则上不予调整和更换。

（七）甲方根据培训课程的需要为学员免费提供预先已经承诺的、基础的、必要的辅助教学资料；除此之外，经征求乙方同意，甲方代购、代为印制、代为拷贝学习资料可以收取合理费用。

（八）如乙方在校期间认真遵守甲方各项管理规定、无违纪违规行为，认真学习（出勤率达85%）却仍未能通过毕业考试的情况下，可在所读课程学习结束之日起一年内向甲方申请重读，甲方应予以安排。甲方根据正常教学计划将重读学员安排在适当班级中进行重读，乙方不应要求甲方单独聘请和安排教师开设新班或单独补课。

（九）除国家相关法律法规和《深圳市民办职业培训机构管理规范》中规定的有关学费退费的情形之外，甲方无需承担其他任何退费义务，乙方也不得以除此以外的任何理由提出退费要求。

   

三、乙方权利和义务 

（一）乙方承诺：在学习期间严格遵守国家的法律和学校的各项规章制度；如出现违法犯罪情况及违反学校规章制度影响学习的情形，不得要求甲方另行安排学习机会，不得要求甲方退还学费及其他费用；如出现严重违纪，甲方可以按学校相关制度的处理。

（二）乙方承诺：学习期间确保上课的出勤率达到85％以上，病假和事假视为出勤，计入出勤率考核中。病假须提供区级以上人民医院的诊断证明；事假需提前三天提交请假单，如有特殊情况无法提前请假，须在上课前以短信或微信方式向班主任请假，并在三日内补写请假条，未在规定时间内补写请假条者视为旷课。三次及以上旷课的，甲方有权开除乙方学籍且不退还乙方所交费用。

（三）乙方应在参加学习前缴纳课程全部学费，在未交齐全款前不得要求甲方安排学习。

（四）乙方承诺严格遵守甲方的具体教学安排，该教学安排包括但不限于乙方的上课地点、时间等，具体安排以甲方的【课程表】为准。

（五）乙方务必保证按时按量的完成甲方课程中明确规定的学习任务；当日的作业或实训任务须在次日或下一次上课前完成。连续三次或累计五次未能按时完成作业及实训任务的，乙方不得要求甲方安排补课、重读、就业推荐等。

（六）乙方知悉并同意，甲方收取的费用仅为培训费。学习过程中所涉及到的教材费、资料费、学习用品及工具费等其他相关费用学员自理（按实际费用实收）。

（七）乙方知悉并同意，如因政策变动等特殊原因乙方需转班，只有当乙方认真遵守甲方教学管理制度时，乙方才可提出申请转班；如乙方出现违反五次以上考勤管理的（包括但不限于旷课、逃课、迟到、早退、不遵守课堂纪律等情况），乙方不得提出转班申请。

    （八）乙方知悉并同意，在学习期间，乙方因故不能上课须按规定请假。若乙方不按规定请假，甲方有权扣除该次课程相应的课时费用。

（九）乙方同意并知悉，甲方提供的内部教材及学习资料仅限乙方本人使用，乙方不得擅自将内部资料告知他人或借给他人使用，否则视为侵犯甲方权利，需承担相应的法律责任。

五、其他事项

（一）乙方参加学习期间有以下情节之一的，不享受本合同第二条第（四）款约定的推荐实习及就业服务： 

1、乙方未通过甲方的毕业考试或毕业设计考核，未取得甲方结业证书。 

2、乙方学习期间未达到甲方规定的85%以上考勤率。

3、乙方拒绝至甲方推荐的专业对口单位实习或就业。 

（二）因本协议所发生的任何纠纷，双方应友好协商解决，协商不成的，任何一方均可向本协议的签订地区有管辖权的人民法院提请诉讼。

（三）如有未尽事宜或对本合同有任何变更，可由甲、乙双方协商一致，签订补充或变更协议予以明确。

 

六、合同生效

此合同一式肆份，甲方持有叁份、乙方持有壹份，自双方签字或盖章之日起生效，具有同等法律效力。
